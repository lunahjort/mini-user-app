# Användarprofil-app

Ett miniprojekt byggt på JavaScript-grunderna.

## Funktioner
- Hämtar användardata från ett externt API (JSONPlaceholder)
- Visar namn, användarnamn, email och telefon
- Knapp-interaktion med eventlistener

## Tekniker
- HTML
- CSS (Flexbox)
- JavaScript (fetch, async/await, DOM-manipulation)
